# Portafolio

Descripción del proyecto:
Este proyecto consiste en el desarrollo de un portafolio web personal creado para presentar de manera profesional mis habilidades, herramientas y experiencia como desarrollador web en formación. El sitio funciona como una tarjeta de presentación digital donde los visitantes pueden conocer más sobre mi perfil, consultar las tecnologías con las que trabajo y visualizar algunos de los proyectos que he realizado durante mi proceso de aprendizaje.

El portafolio cuenta con una sección de presentación personal, una lista de habilidades y herramientas, un apartado de proyectos destacados con enlaces a las demostraciones en vivo y sus respectivos repositorios de GitHub, además de una sección de contacto con enlaces a mis redes sociales. El diseño fue desarrollado tomando como referencia un brief realizado en Figma y adaptado para diferentes tamaños de pantalla, buscando ofrecer una experiencia visual clara, moderna y fácil de navegar tanto en dispositivos de escritorio como en tablets y teléfonos móviles.

El proyecto también me permitió poner en práctica conocimientos de desarrollo web, diseño responsivo, organización del código y control de versiones, creando una página que puede continuar creciendo conforme adquiera nuevas habilidades y desarrolle más proyectos.

Tecnologías utilizadas
1.-HTML para estructurar semánticamente el contenido y las diferentes secciones del portafolio.
2.-CSS para crear el diseño visual, los estilos, colores, tipografías y efectos de interacción.
3.-Flexbox y CSS Grid para organizar y distribuir los elementos de la página de manera flexible.
4.-Media Queries para adaptar el diseño a diferentes tamaños de pantalla y crear una experiencia responsiva.
5.-Posicionamiento CSS (relative y absolute) para organizar elementos superpuestos, como los iconos de tecnologías sobre las imágenes de los proyectos.
6.-Gradientes y pseudo-elementos (::after) para crear efectos visuales como el subrayado degradado de los enlaces.
7.-Git y GitHub para el control de versiones, almacenamiento y gestión del proyecto.
8.-GitHub Pages para publicar el portafolio y permitir su visualización en línea.

Planes de mejora del proyecto:

Como mejoras futuras, me gustaría continuar actualizando el portafolio conforme avance en mi formación como desarrollador. Algunas ideas son agregar nuevos proyectos personales y profesionales, incluir una sección con mis certificaciones y cursos realizados, incorporar un botón funcional para descargar mi CV, agregar más información sobre las tecnologías que voy aprendiendo y mejorar la presentación de cada proyecto mediante imágenes, descripciones y enlaces.
