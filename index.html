Página web de una cafetería

Triple Espresso una página web desarrollada para una cafetería, cuyo objetivo es ofrecer una experiencia agradable y sencilla para los usuarios. El sitio presenta información sobre la cafetería, una sección para realizar reservaciones mediante un formulario y un pie de página con enlaces a redes sociales. El diseño está pensado para ser limpio, atractivo y fácil de navegar, siguiendo las especificaciones proporcionadas en el proyecto.
